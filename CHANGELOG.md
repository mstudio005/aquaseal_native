# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-28
### Added
- Initial stable Android release (AquaSeal v1.0.0)
- Embedded Python runtime (Chaquopy) and yt-dlp for on-device downloads
- Downloads page (history, filters, open/delete actions)
- Native `FileOpener` plugin to open files via device apps or in-app player
- Toasts and confirm dialog UI for improved UX
- Multi-language UI support

### Changed
- Project licensing updated to GPL-3.0-or-later
- Documentation consolidated and updated for Android-first workflow

### Notes
- APK produced at `android/app/build/outputs/apk/debug/app-debug.apk` during debug builds.
