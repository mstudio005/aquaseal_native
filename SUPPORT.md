# Support Guide

Need help with AquaSeal? Use the channels below.

## 1. Self-service

- Read the main [`README.md`](README.md) for setup, architecture, and release steps.
- Review [`CHANGELOG.md`](CHANGELOG.md) for recent fixes or breaking changes.
- Check [`docs/LOCALIZATION.md`](docs/LOCALIZATION.md) and [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) for specialty topics.

## 2. Report a bug

- Use the "Bug report" issue template under `.github/ISSUE_TEMPLATE/`.
- Include logs (from Logcat), device model, Android version, and reproduction steps.

## 3. Request a feature

- Open a "Feature request" issue describing the use case and desired UX.
- Attach wireframes or mockups if available.

## 4. Security / privacy concerns

- Follow the process outlined in [`SECURITY.md`](SECURITY.md).

## 5. Questions & discussions

- For general questions, open a Discussion or issue tagged `question`.
- Commercial support is not available yet; contributions from the community keep the project moving.
